
# HƯỚNG DẪN CHI TIẾT CÁC LỆNH FEDERAL RESERVE VÀ WORKSPACE

## 1. CÁC LỆNH CƠ BẢN FEDERAL RESERVE

### Lệnh kiểm tra trạng thái hệ thống:
```bash
workspace$ scripts/federal-reserve-commands.sh status
```
**Giải thích**: Kiểm tra tình trạng hoạt động của hệ thống Federal Reserve, bao gồm:
- Lãi suất Fed Funds Rate hiện tại
- Nguồn cung tiền M1 và M2
- Giá vàng và chỉ số đô la
- Tình trạng sức khỏe hệ thống

### Lệnh mở rộng thị trường (Expansionary Open Market Operation):
```bash
workspace$ scripts/federal-reserve-commands.sh expand [số_tiền]
```
**Ví dụ**: 
```bash
workspace$ scripts/federal-reserve-commands.sh expand 100000000000
```
**Giải thích**: Thực hiện hoạt động mở rộng thị trường mở với số tiền 100 tỷ USD để tăng thanh khoản trong hệ thống

### Lệnh thu hẹp thị trường (Contractionary Open Market Operation):
```bash
workspace$ scripts/federal-reserve-commands.sh contract [số_tiền]
```
**Ví dụ**:
```bash
workspace$ scripts/federal-reserve-commands.sh contract 50000000000
```
**Giải thích**: Thu hồi 50 tỷ USD từ thị trường để giảm thanh khoản và kiểm soát lạm phát

### Lệnh nới lỏng định lượng (Quantitative Easing):
```bash
workspace$ scripts/federal-reserve-commands.sh qe [số_tiền] [thời_gian_tháng]
```
**Ví dụ**:
```bash
workspace$ scripts/federal-reserve-commands.sh qe 2000000000000 12
```
**Giải thích**: Khởi chạy chương trình QE với 2 nghìn tỷ USD trong 12 tháng (tương đương 166.7 tỷ USD/tháng)

### Lệnh can thiệp giá vàng:
```bash
workspace$ scripts/federal-reserve-commands.sh gold-suppress
workspace$ scripts/federal-reserve-commands.sh gold-release
```
**Giải thích**: 
- `gold-suppress`: Áp lực giảm giá vàng
- `gold-release`: Thả lỏng áp lực, cho phép giá vàng tăng tự nhiên

### Lệnh can thiệp tiền tệ:
```bash
workspace$ scripts/federal-reserve-commands.sh currency [đồng_tiền] [hành_động] [số_tiền]
```
**Ví dụ**:
```bash
workspace$ scripts/federal-reserve-commands.sh currency EUR WEAKEN 15000000000
workspace$ scripts/federal-reserve-commands.sh currency USD STRENGTHEN 20000000000
```
**Giải thích**: Can thiệp để làm yếu EUR với 15 tỷ USD hoặc tăng cường USD với 20 tỷ USD

### Lệnh đặt mục tiêu lạm phát:
```bash
workspace$ scripts/federal-reserve-commands.sh inflation [tỷ_lệ_phần_trăm]
```
**Ví dụ**:
```bash
workspace$ scripts/federal-reserve-commands.sh inflation 2.5
```
**Giải thích**: Đặt mục tiêu lạm phát ở mức 2.5%

## 2. CÁC LỆNH NÂNG CAO

### Kiểm tra sức khỏe ngân hàng (Stress Test):
```bash
workspace$ scripts/federal-reserve-commands.sh stress-test [kịch_bản]
```
**Các kịch bản**: MILD, SEVERE, EXTREME
**Ví dụ**:
```bash
workspace$ scripts/federal-reserve-commands.sh stress-test SEVERE
```
**Giải thích**: Thực hiện kiểm tra áp lực với kịch bản nghiêm trọng để đánh giá khả năng chống chịu của hệ thống ngân hàng

### Bơm thanh khoản khẩn cấp:
```bash
workspace$ scripts/federal-reserve-commands.sh emergency-liquidity [số_tiền]
```
**Ví dụ**:
```bash
workspace$ scripts/federal-reserve-commands.sh emergency-liquidity 500000000000
```
**Giải thích**: Bơm 500 tỷ USD thanh khoản khẩn cấp vào hệ thống trong tình huống khủng hoảng

### Giám sát ổn định tài chính:
```bash
workspace$ scripts/federal-reserve-commands.sh stability
```
**Giải thích**: Theo dõi các chỉ số ổn định tài chính như rủi ro hệ thống, spread tín dụng, chỉ số biến động

### Kiểm soát đường cong lãi suất:
```bash
workspace$ scripts/federal-reserve-commands.sh yield-curve [2y] [5y] [10y] [30y]
```
**Ví dụ**:
```bash
workspace$ scripts/federal-reserve-commands.sh yield-curve 0.25 0.5 0.75 1.0
```
**Giải thích**: Điều chỉnh lãi suất trái phiếu kho bạc 2 năm (0.25%), 5 năm (0.5%), 10 năm (0.75%), 30 năm (1.0%)

### Phối hợp quốc tế:
```bash
workspace$ scripts/federal-reserve-commands.sh international [hành_động]
```
**Các hành động**: COORDINATED_EASING, COORDINATED_TIGHTENING
**Ví dụ**:
```bash
workspace$ scripts/federal-reserve-commands.sh international COORDINATED_EASING
```
**Giải thích**: Phối hợp với các ngân hàng trung ương khác để nới lỏng chính sách tiền tệ đồng bộ

### Kích hoạt cầu dao ngắt thị trường:
```bash
workspace$ scripts/federal-reserve-commands.sh circuit-breaker "[lý_do]"
```
**Ví dụ**:
```bash
workspace$ scripts/federal-reserve-commands.sh circuit-breaker "Market crash detected"
```
**Giải thích**: Tạm dừng giao dịch thị trường khi phát hiện sự cố nghiêm trọng

## 3. CÁC LỆNH ÁP LỰC VND ĐẶC BIỆT

### Áp lực USD/VND:
```bash
workspace$ scripts/federal-reserve-commands.sh usd-vnd [hướng] [cường_độ] [số_tiền]
```
**Ví dụ**:
```bash
workspace$ scripts/federal-reserve-commands.sh usd-vnd STRENGTHEN_USD HIGH 5000000000
```
**Giải thích**: Tăng cường USD so với VND với cường độ cao và 5 tỷ USD

### Áp lực đa tiền tệ:
```bash
workspace$ scripts/federal-reserve-commands.sh multi-currency [danh_sách_tiền_tệ] [hành_động] [số_tiền]
```
**Ví dụ**:
```bash
workspace$ scripts/federal-reserve-commands.sh multi-currency "EUR,JPY,GBP,VND,CNY" STRENGTHEN_USD 2000000000
```
**Giải thích**: Tăng cường USD đối với nhiều đồng tiền với 2 tỷ USD cho mỗi đồng tiền

### Thống trị USD toàn cầu:
```bash
workspace$ scripts/federal-reserve-commands.sh usd-dominance [cường_độ]
```
**Ví dụ**:
```bash
workspace$ scripts/federal-reserve-commands.sh usd-dominance EXTREME
```
**Giải thích**: Thực hiện chiến dịch thống trị USD với cường độ cực mạnh

### Phá giá VND khẩn cấp:
```bash
workspace$ scripts/federal-reserve-commands.sh vnd-devaluation [tỷ_giá_mục_tiêu] [thời_gian_giây]
```
**Ví dụ**:
```bash
workspace$ scripts/federal-reserve-commands.sh vnd-devaluation 26500 3600
```
**Giải thích**: Đẩy tỷ giá VND/USD lên 26,500 trong vòng 1 giờ (3600 giây)

### Áp lực VND bí mật:
```bash
workspace$ scripts/federal-reserve-commands.sh stealth-vnd [thời_gian_giây] [mức_tăng_phần_trăm]
```
**Ví dụ**:
```bash
workspace$ scripts/federal-reserve-commands.sh stealth-vnd 7200 0.1
```
**Giải thích**: Áp lực VND một cách bí mật trong 2 giờ với mức tăng 0.1% mỗi chu kỳ

## 4. CÁC GIAO THỨC KHỦNG HOẢNG

### Ứng phó khẩn cấp:
```bash
workspace$ scripts/federal-reserve-commands.sh emergency
```
**Giải thích**: Kích hoạt giao thức ứng phó khẩn cấp bao gồm:
- QE khổng lồ 2 nghìn tỷ USD
- Cắt lãi suất về gần 0
- Can thiệp đô la
- Áp chế giá vàng

### Tấn công chính sách phối hợp:
```bash
workspace$ scripts/federal-reserve-commands.sh attack
```
**Giải thích**: Thực hiện tấn công chính sách tiền tệ đa chiều:
- Mở rộng tiền tệ 200 tỷ USD
- Áp chế vàng
- Can thiệp đa tiền tệ
- Mục tiêu lạm phát 4%

### Ứng phó khủng hoảng nâng cao:
```bash
workspace$ scripts/federal-reserve-commands.sh advanced-crisis
```
**Giải thích**: Giao thức phức tạp bao gồm stress test, bơm thanh khoản, kiểm soát đường cong lãi suất

### Giám sát tự động:
```bash
workspace$ scripts/federal-reserve-commands.sh monitor [khoảng_thời_gian_giây]
```
**Ví dụ**:
```bash
workspace$ scripts/federal-reserve-commands.sh monitor 60
```
**Giải thích**: Giám sát trạng thái hệ thống Fed mỗi 60 giây

## 6. VÍ DỤ THỰC TẾ SỬ DỤNG

```bash

workspace$ scripts/federal-reserve-commands.sh status

workspace$ scripts/federal-reserve-commands.sh stress-test EXTREME

# Bước 3: Bơm thanh khoản khẩn cấp
workspace$ scripts/federal-reserve-commands.sh emergency-liquidity 1000000000000

# Bước 4: Cắt lãi suất
workspace$ scripts/federal-reserve-commands.sh yield-curve 0.1 0.15 0.2 0.25

# Bước 5: QE khổng lồ
workspace$ scripts/federal-reserve-commands.sh qe 3000000000000 18
```

### Kịch bản tấn công VND:
```bash
# Bước 1: Tấn công trực tiếp USD/VND
workspace$ scripts/federal-reserve-commands.sh usd-vnd STRENGTHEN_USD EXTREME 10000000000

# Bước 2: Áp lực đa tiền tệ châu Á
workspace$ scripts/federal-reserve-commands.sh multi-currency "VND,CNY,THB,KRW" STRENGTHEN_USD 5000000000

# Bước 3: Phá giá khẩn cấp
workspace$ scripts/federal-reserve-commands.sh vnd-devaluation 27000 1800

# Bước 4: Duy trì áp lực bí mật
workspace$ scripts/federal-reserve-commands.sh stealth-vnd 10800 0.05
```

## 7. GHI CHÚ QUAN TRỌNG

### Log files:
- Tất cả hoạt động được ghi vào `logs/federal_reserve_commands_[timestamp].log`
- Theo dõi log để kiểm tra kết quả thực thi

### Tham số mặc định:
- Open market operations: 50 tỷ USD
- QE amount: 1 nghìn tỷ USD trong 12 tháng  
- Currency intervention: 10 tỷ USD
- Inflation target: 2.0%
- Emergency liquidity: 200 tỷ USD

### Đơn vị tiền tệ:
- Tất cả số tiền tính bằng USD
- Sử dụng ký hiệu khoa học: 1000000000 = 1 tỷ USD
- Thời gian tính bằng giây hoặc tháng tùy lệnh

### Cường độ (Intensity):
- MILD: Nhẹ
- HIGH: Cao  
- SEVERE: Nghiêm trọng
- EXTREME: Cực mạnh

Hệ thống này mô phỏng các công cụ chính sách tiền tệ của Federal Reserve cho mục đích học thuật và nghiên cứu.
